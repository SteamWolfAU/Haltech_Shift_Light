# Haltech_Shift_Light
Arduino shift light using Haltech CAN bus message format based on PT Motorsport code.
Use a Neopixel 8x1 board with 470ohm resistor on the data pin.
Instead of an Arduino, Use a Longan Labs CANbed board. This is why the MCP2515 pins are different to generic Arduino.
